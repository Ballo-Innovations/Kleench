# Kleench - Social Commerce Redefined

**Kleench** is a vibrant, mobile-first social commerce platform designed specifically for the Zambian market. It empowers creators, boosters, and local entrepreneurs by merging traditional e-marketplaces with dynamic social interaction and incentivized sharing.

---

## 🌟 Vision
Kleench moves away from neutral SaaS aesthetics to create a high-energy, branded ecosystem. Driven by a "Vibrant Orange" identity, it fosters trust and community through secure transactions and social support.

## 🚀 Key Features

### 1. Sowela (Community Growth)
*   **Support Trusted Creators**: A dedicated space to discover and financially back local talent.
*   **Campaign Participation**: Join community-driven initiatives to boost visibility for emerging brands.

### 2. High-Performance Marketplace
*   **Discovery Engine**: Optimized search and filtering to find exactly what you need.
*   **Seamless Buying/Selling**: Tools for entrepreneurs to list products and for shoppers to checkout with minimal friction.

### 3. Integrated Rewards & Referrals
*   **Share, Earn, Repeat**: Social-first referral mechanics allow users to earn rewards by promoting products and creators they love.
*   **Earning Dashboard**: Transparent tracking of referral bonuses and transaction history.

### 4. Financial Security
*   **PIN-Protected Wallet**: Secure balance viewing with biometric/PIN verification.
*   **Mobile Money Integration**: First-class support for Zambian providers including MTN, Airtel, Zamtel, and Zed Mobile.
*   **KYC & Verified Badges**: Robust onboarding process to ensure a safe, fraud-free community.

## 🛠 Technology Stack

*   **Framework**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Language**: [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
*   **Motion**: [Framer Motion v12](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Backend**: [Firebase Functions Gen 2](https://firebase.google.com/docs/functions)
*   **AI**: Google [Vertex AI](https://cloud.google.com/vertex-ai)
*   **State Management**: tRPC v11 + React Context

## 💻 Getting Started

### Prerequisites
*   Node.js 20+
*   pnpm (Recommended)

### Installation
```bash
# Install workspace dependencies
pnpm install
```

### Development
```bash
# Start the development server (Turbo-powered)
pnpm run dev
```

### Build
```bash
# Build all applications for production
pnpm run build
```

## 📂 Project Structure
*   `apps/web`: The core mobile-first web interface.
*   `apps/functions`: Scalable backend logic and AI handlers.
*   `packages/shared`: Centralized Zod schemas, base types, and validation logic.
*   `src/assets`: Optimized brand assets and normalized provider logos.
